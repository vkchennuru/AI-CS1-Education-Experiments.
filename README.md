# AI-Code-Judgment-CS1

This repository contains the benchmark dataset, prompts, evaluation scripts, and experimental artifacts for the research paper:

**Beyond Code Generation: Developing Code Judgment Skills in Introductory Programming**

## Repository Structure

- dataset/ : CS1 benchmark problems
- prompts/ : Standard prompt used for all experiments
- evaluation/ : Automated evaluation scripts (to be added)
- notebooks/ : Google Colab notebook (to be added)
- results/ : Experimental results (to be added)

This repository accompanies our ACM COMPUTE 2026 submission.
